```YAML
{

    'key': 'lorem',

    'region_name': 'lorem' 
  },
  'flow': {
    'api_key': 'lorem',
    'workflow_id': 'lorem',
    'base_url': 'https://api.workflowchef.ai'
  }
}
```
Change 'lorem' with specific value or contact me for result.
